# Robotics 101: The Essentials of Automation

This repository contains the project and presentation materials developed during my Bachelor's studies, focusing on the foundational concepts of robotics and automation.

## 📌 Project Overview
An entry-level comprehensive study on industrial robotics, core components, and real-world automation applications.

### Key Highlights:
- **Core Components covered:** Sensors, Effectors, Actuators, Controllers, and Robotic Arms.
- **Classification studied:** Mobile, Stationary, Autonomous, Remote-Controlled, and Virtual robots.
- **Ethics:** Analysis of Asimov's Zeroth Law in modern automation.
